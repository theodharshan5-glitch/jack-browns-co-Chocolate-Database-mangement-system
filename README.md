# Jack Browns & Co – Dynamic DBMS

A Class 12 CBSE Computer Science project.

This project is a dynamic database management system for a chocolate factory named **Jack Browns & Co**.

Built using:

- Python
- PyQt5
- MySQL

## Features

- Dynamic table selection
- Load any table automatically
- View records in tabular format
- Edit cells directly
- Add rows
- Delete rows
- Save changes to MySQL database
- Modern dark UI
